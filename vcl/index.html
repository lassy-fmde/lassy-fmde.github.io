About

This is the web-site of the Visual Contract Language (VCL). VCL is a visual language for the abstract specification of software designs. It is being developed at the LASSY laboratory of the University of Luxembourg. This site is a public repository for VCL. It makes available papers, technical reports and VCL's tool.

The Visual Contract Builder (VCB), VCL's tool, is an Eclipse plugin. It is currently under development. You can try the current version of VCB. A tutorial on how to use VCL and VCB can be found here .
